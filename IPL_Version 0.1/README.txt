Interactive Port Layout - Installation Guide
============================================

1. Open the Installer folder.
2. Double-click the Setup EXE:
   InteractivePortLayout_Setup_20260418_084604.exe
3. Follow the installer steps.
4. Launch "Interactive Port Layout" from Start Menu or Desktop shortcut.

Notes:
- Internet access is required for live map tiles.
- If prompted, install Microsoft Edge WebView2 Runtime.
